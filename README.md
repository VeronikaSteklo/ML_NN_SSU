# Машинное обучение

## 1 задание

Реализация основных алгоритмов машинного обучения (регрессия, ансамбли, кластеризация) на языке Python с использованием
библиотеки NumPy. Также было проведено сравнение кастомных решений с эталонными реализациями из библиотеки Scikit-learn.

Реализованные методы:
1. KNN
2. Линейная и полиномиальные регрессии
3. Дерево решений
4. Ансамбли
5. Алгоритмы кластеризации (k-means и DBSCAN)
